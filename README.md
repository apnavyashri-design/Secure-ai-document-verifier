About Secure AI Document Verifier
    A simple AI + blockchain based web application that verifies the authenticity of uploaded documents using a Python backend and a minimal web frontend.

Features
- Upload document files through a web interface.
- Run automated checks using AI or rule-based logic.
- Generate a verification result for each document.
- Basic blockchain-style idea using hashes/immutability for verification records.[web:48]

 Tech Stack
- Python (backend, e.g., Flask or FastAPI)
- HTML, CSS (frontend)
- GitHub for version control and collaboration

Project Structure
- `app.py` – Main backend application (routes and verification logic).
- `frontend.html` – User interface for uploading documents and viewing results.
- `styles.css` – Styling for the web page layout and components.
